# Advanced-AI-GitHub-PR-Code-Reviewer
This project is an AI-powered GitHub Pull Request review system that automatically analyzes PRs using GitHub Webhooks and a multi-agent LangGraph pipeline powered by GPT-4o-mini. It runs parallel agents for static analysis, security, architecture, and code style reviews, generating structured feedback. Built with FastAPI microservices, Celery, Redis, PostgreSQL, Prometheus, Grafana, and Langfuse, the system also includes a self-learning service that improves review quality over time.
